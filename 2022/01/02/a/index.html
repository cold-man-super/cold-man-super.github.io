hexo

发表于2022-01-02更新于2022-01-02阅读次数

1 所需环境

1.1 有一个GitHub账号

1.2 安装了node.js、npm

1.3 安装了git

2 搭建github仓库

新建一个名为：用户名.github.io的仓库
ps:用户名一定与github的用户名一致，注意后缀

3 配置SSH

输入：
cd ~/. ssh
ssh-keygen -t rsa -C "邮箱"
然后连续3次回车，最终会生成一个文件在用户目录下，打开用户目录，找到.ssh\id_rsa.pub文件，复制里面的内容，打开github，设置 -> SSH and GPG keys -> New SSH key：
将复制的内容粘贴进去就ok，title随意

4 hexo

用windows cmd和git bash都可以，我用的是后者
输入
npm install -g hexo

进行安装hexo
然后
hexo init
hexo g
hexo s
打开浏览器访问 http://localhost:4000 即可看到内容
ps:切记不能按Ctrl C来复制，因为按下是退出，会访问不了
如果还是访问不了可以参考这篇文章
https://blog.csdn.net/weixin_39691535/article/details/86663749

5 上传到github

_config.yml中最后加入：
deploy:
type: git
repository: git@github.com:用户名/用户名.github.io.git
branch: master
在url那一行也要输入：https://用户名.github.io
在输入：
npm install hexo-deployer-git --save
下载一个插件

6 访问

如果能够上传到github上面，那么输入
用户名.github.io
便可以访问hexo博客界面了
ps:要将master分支改为默认

7 更换主题

在
https://hexo.io/themes/
可以选择喜欢的主题，然后根据主题的说明去下载和配置
